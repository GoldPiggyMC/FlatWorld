<h1 align="center">Flat World</h1>

## What is Flat World?

This is a mod which add the Vanilla superflat world.

## How to visit the flat world?

- Build the Nether portal but use stone instead of obsidian.
- Activate it by using the iron ingot.
- Enjoy.
